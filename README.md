# Event Tracker Bot — инструкция по развёртке на Railway

## Шаг 1. Создать Telegram-бота

1. Открыть Telegram → найти **@BotFather**
2. Отправить `/newbot`
3. Придумать имя и username (например, `MyEventTrackerBot`)
4. BotFather пришлёт **BOT_TOKEN** — сохранить его

## Шаг 2. Узнать свой CHAT_ID

1. Написать боту **@userinfobot** в Telegram
2. Он пришлёт твой **id** — это и есть CHAT_ID

## Шаг 3. Развернуть на Railway

1. Зарегистрироваться на **railway.app** (бесплатно, даётся $5/мес)
2. Нажать **"New Project"** → **"Deploy from GitHub repo"**
3. Загрузить файлы бота в GitHub-репозиторий (или использовать Railway CLI):
   ```
   railway login
   railway init
   railway up
   ```
4. В Railway открыть проект → вкладка **"Variables"** → добавить:
   - `BOT_TOKEN` = токен от BotFather
   - `CHAT_ID` = твой ID из @userinfobot
5. Railway автоматически запустит `worker: python bot.py`

## Шаг 4. Синхронизировать события

1. Открыть веб-приложение Event Tracker
2. Нажать кнопку **"Синк с ботом"** (скопирует команду в буфер)
3. Вставить в чат с ботом в Telegram
4. Бот ответит: ✅ Синхронизировано N событий

## Команды бота

| Команда | Описание |
|---------|----------|
| `/today` | События на сегодня |
| `/tomorrow` | События на завтра |
| `/upcoming` | Ближайшие 7 дней |
| `/list` | Все предстоящие события |
| `/sync <JSON>` | Синхронизация из веб-приложения |

## Формат уведомлений

**За день до события (в настроенное время):**
```
⚠️ Завтра (15.06):

🔺 Встреча с инвестором · 10:00 · Иван Петров [Встреча]
🔸 Сдать отчёт · 18:00 [Дедлайн]
```

**В день события (в настроенное время):**
```
🔴 Сегодня (15.06):

🔺 Встреча с инвестором · 10:00 · Иван Петров
   📝 Обсудить раунд B, подготовить deck
```

## Изменить время напоминаний

В веб-приложении: **Настройки** → изменить время → нажать "Синк с ботом" → отправить боту.
Бот автоматически перезапустит scheduler с новым временем при следующем деплое.
